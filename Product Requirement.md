# SMITHS-PERSION - Product Requirement Document

## 1. Patient Management

* Register patient data
* Define surgical urgency (Emergency / Elective)
* Assign surgical procedure
* Calculate surgery scoring

### Data Fields:

* Patient ID
* Name
* Gender
* Date of Birth
* Age
* Height / Weight
* Blood Type
* Patient Type (Adult / Pediatric / Neonate)
* Infectious Status
* Patient Class
* Diagnosis (Preoperative & Proposed)

---

## 2. Surgery Scheduling

* Multi-surgery selection
* Auto surgeon assignment
* Auto scheduling time
* Sorting & filtering schedule

### Features:

* Daily / Weekly / Monthly scheduling
* Operating room assignment
* Gantt chart visualization

---

## 3. Surgery Execution Tracking

### Pre-operative

* Preparation
* Patient identification
* Supporting examination

### Anesthesia

* Anesthesia delivery
* Intubation / ventilation
* Vital monitoring

### Intra-operative

* Surgery execution tracking

### Post-operative

* Recovery room transfer

---

## 4. Surgical History

* Patient history tracking
* Surgery detail review
* Action log tracking (time-based)

---

## 5. Security

* Access code for sensitive data
* Password verification before viewing

---

## 6. Configuration

* User management
* Operating room management
* Hospital configuration
* Surgical menu management
